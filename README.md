# District Wars

> Build. Battle. Conquer.

A 4-player real-time strategy game playable in the browser. Control a district, build your economy, train your army, and conquer all four flags to win.

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

- **Engine:** Phaser.js 3
- **Language:** TypeScript
- **Build:** Vite
- **Server:** Node.js + Socket.io
- **Deploy:** Vercel
